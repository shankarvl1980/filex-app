FileX Manager

Cloud file explorer for Android. This page covers Microsoft OneDrive / Microsoft Graph integration used by the app.

What the app does

  • Connects to your Microsoft account after you sign in
  • Lets you browse OneDrive folders/files
  • Upload, download, rename, move, and delete files/folders you select

Permissions requested

The OneDrive connector uses Microsoft Graph delegated permissions:

  • Files.ReadWrite – manage your OneDrive files/folders (upload/download/rename/delete/move)
  • User.Read – basic profile info for sign-in
  • offline_access – keep you signed in (refresh tokens)

All actions occur as the signed-in user and only when initiated from the app UI.

Data handling summary

  • We access file metadata/content only to perform the operations you request.
  • Access tokens are stored on the device and protected using Android security mechanisms.
  • We do not sell your data.
