Privacy Statement

Last updated: 2026-02-21

Scope

This Privacy Statement describes how FileX Manager handles data when you connect Microsoft OneDrive using Microsoft Graph.

Data we access

  • Basic profile data required for sign-in (e.g., user identifier)
  • OneDrive file/folder metadata (names, paths, sizes, timestamps) as needed for browsing
  • File content only when you upload/download/open a file in the app

How we use data

  • To show your OneDrive folder structure and file list
  • To perform actions you request: upload, download, rename, move, delete
  • To maintain your signed-in session (token refresh)

Storage and security

  • Authentication tokens are stored on-device and protected using Android security mechanisms.
  • We do not intentionally store your OneDrive file content on our servers.
  • Network traffic to Microsoft Graph uses HTTPS.

Sharing

We do not sell your personal data. We do not share your OneDrive content with third parties except as required to provide the service (Microsoft Graph APIs you choose to access).
